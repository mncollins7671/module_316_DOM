# DOM Fundamentals Course

This course prepares you for the SBA 316: The Document Object Model assessment. Each lesson builds upon the previous one, giving you all the skills needed to complete the final project.

## Course Structure

| Lesson | Topic | Key Concepts |
|--------|-------|--------------|
| 01 | Introduction to the DOM | What is the DOM, DOM tree structure |
| 02 | Selecting Elements | getElementById, querySelector, querySelectorAll |
| 03 | DOM Traversal | Parent-child-sibling relationships |
| 04 | Iterating Over Collections | Looping through NodeLists and HTMLCollections |
| 05 | Creating & Adding Elements | createElement, appendChild, prepend |
| 06 | Templating | DocumentFragment, cloneNode, HTML templates |
| 07 | Modifying Content | innerHTML, innerText, textContent |
| 08 | Styles and Classes | style property, classList API |
| 09 | Modifying Attributes | setAttribute, getAttribute, data attributes |
| 10 | Event Handling | addEventListener, event types, event objects |
| 11 | Browser Object Model | window, location, history, navigator |
| 12 | HTML Form Validation | Built-in validation attributes |
| 13 | DOM Event Validation | JavaScript-based form validation |
| 14 | Capstone Project | Putting it all together |

## How to Use This Course

1. Read through each `lesson.md` file
2. Complete the exercises in the `exercises/` folder
3. Check your work against the `solutions/` folder
4. Move on to the next lesson

## Final Assessment

After completing all lessons, you'll be ready to tackle the SBA. See [sba.md](sba.md) for full requirements.
